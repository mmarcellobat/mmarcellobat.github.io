JS: Tipos de dados

  • Número: qualquer número, incluindo números com casas decimais: 8, 1516, 23.42.
  • String: Qualquer agrupamento de caracteres no teclado (letras, números, espaços, símbolos etc.) entre aspas apóstofres: '...' ou aspas "...".
  • Booleano: esse tipo de dados tem apenas dois valores possíveis - true ou false (sem aspas).
  • Nulo: Esse tipo de dados representa a ausência intencional de um valor e é representado pela palavra-chave null (sem aspas).
  • Indefinido: esse tipo de dado é denotado pela palavra-chave undefined (sem aspas). Também representa a ausência de um valor quando não inicializada a variável, embora tenha um uso diferente de nulo.

JS: Número

Para saber mais sobre como o computador registra números inteiros: Sistema de numeração binário e Complemento de 2 (inteiros negativos).

Para saber mais sobre como o computador registra números de ponto flutuante: Sobre norma IEEE 754 e Conversor binário para ponto flutuante.

Ao contrário de diversas outras linguagens, JS considera todos os números como um único tipo (inteiros, ponto flutuante).

Veja: propriedades e métodos para Number

Propriedades

  • Number.NaN

    • Not a Number, não é um número; usado para indicar a impossibilidade matemática de se calcular algo.

  • Number.EPSILON

    • Na computação, é o menor número que somado a 1 produza resultado diferente de 1, ou seja, que não é arredondado; veja aqui os detalhes: Épsilon de máquina

Métodos

  • Number.isNaN()

    • Testa se o valor é NaN; exemplo: console.log(Number.isNaN(0/0)); // true

  • Number.isInteger()

    • Testa se o valor é inteiro.

  • Number.parseInt()

    • A partir de uma string, retorna um número inteiro.

  • Number.parseFloat()

    • A partir de uma string, retorna um número de ponto flutuante.

JS: Operadores aritméticos

  • Adicionar: +

    • Exemplo: console.log(3 + 2.5); // 5.5

  • Subtrair: -

    • Exemplo: console.log(3 - 2.5); // 0.5

  • Multiplicar: *

    • Exemplo: console.log(37 * 13); // 481

  • Dividir: /

    • Exemplo: console.log(37 / 13); // 2.84615

  • Resto: %

    • Exemplo: console.log(37 % 13); // 11

JS: String

Ao contrário de outras linguagens, o JS não faz distinção entre strings entre aspas simples e strings com aspas duplas; portanto, as seqüências de escape funcionam em strings criadas com apóstrofes ou aspas.

Exemplos de caractere de escape: \' (apóstrofe), \" (aspa), \\ (barra invertida), \n (nova linha), \t (tabulação).

Veja: propriedades e métodos para String

Propriedades

  • ''.length

    • Exemplo: console.log('Amor'.length); // 4

Métodos

  • ''.charAt()

    • Exemplo: console.log('Amor'.charAt(1)); // 'm'

  • ''.toUpperCase() ou ''.toLocaleUpperCase()

    • Exemplo: console.log('aMor'.toUpperCase()); // 'AMOR'

  • ''.toLowerCase() ou ''.toLocaleLowerCase()

    • Exemplo: console.log('ÓdIO'.toLocaleLowerCase('pt-BR')); // 'ódio'

  • ''.startsWith()

    • Exemplo: console.log('Amor'.startsWith('A')); // true

  • ''.includes()

    • Exemplo: console.log('Mississipi'.includes('sissi')); // true

  • ''.endsWith()

    • Exemplo: console.log('Amor'.endsWith('A')); // false

  • ''.replace()

    • Exemplo: console.log('Amor'.replace('mo', 'rmado')); // 'Armador'

  • ''.trim()

    • Exemplo: console.log(' O cavalo chegou '.trim()); // 'O cavalo chegou'

  • ''.slice()

    • Exemplo:

    var str = 'The quick brown fox jumps over the lazy dog.';
console.log(str.slice(31)); // 'the lazy dog.'
console.log(str.slice(4, 19)); // 'quick brown fox'
console.log(str.slice(-4)); // 'dog.'
console.log(str.slice(-9, -5)); // 'lazy'

Concatenação de Strings

Em JS, o operador + consegue concatenar strings. Por exemplo:

var pai = 'José';
var mae = 'Maria';
var txt = 'O pai é ' + pai;
var idade = 16;
txt += ' e a mãe é ' + mae + '!';
console.log(txt + ' Sua idade é ' + idade); // 'O pai é José e a mãe é Maria! Sua idade é 16'

JS: Variáveis

Palavras reservadas:

  • var, escopo de função, pode ser alterada;
  • let, escopo de bloco, pode ser alterada; e
  • const, não pode ser alterada.

Operadores de atribuição matemáticos, incremento e decremento

  • Adicionar e atribuir: +=

    • Exemplo: var x = 150; x += 30; // x = 180

  • Subtrair e atribuir: -=

    • Exemplo: var x = 150; x -= 30; // x = 120

  • Multiplicar e atribuir: *=

    • Exemplo: var x = 150; x *= 30; // x = 4500

  • Dividir e atribuir: /=

    • Exemplo: var x = 150; x /= 30; // x = 5

  • Resto e atribuir: %=

    • Exemplo: var x = 150; x %= 30; // x = 0

Operador typeof

JS: Condicional

Palavras reservadas: if, else if e else

Operadores de comparação

  • Menor que: <
  • Maior que: >
  • Menor ou igual a: <=
  • Maior que ou igual a: >=
  • É igual a (igualdade): ==
  • Não é igual a (não igualdade): !=
  • É igual a (identidade): ===
  • Não é igual a (não identidade): !==

ATENÇÃO!Operadores baseados em igualdade tentarão tornar os tipos de dados iguais antes de fazer a comparação, por exemplo console.log(1 == '1.0'); // true. Por outro lado, operadores baseados em identidade requerem que os dois tipos de dados sejam os mesmos de um pré-requisito, por exemplo console.log(1 === '1.0'); // false.

Operadores lógicos

  • Conjunção (operador e; AND): &&
  • Disjunção (operador ou; OR): ||
  • Negação (operador não; NOT): !

Veracidade e falsidade de variáveis

Você pode checar a existência de uma variável com if (nome-da-variavel). Se der true, a variável existe e possui valor identificável. Será false nas seguintes situações:
  • A variável tem valor 0;
  • A variável é uma string vazia '' ou "";
  • A variável tem valor null (sem valor);
  • A variável tem tipo undefined (não inicializada);
  • A variável tem valor NaN (não é um número);

Operador ternário [ -- ? -- : -- ]

Exemplo:

if (estaCorreto) {
    console.log('Correto!');
} else {
    console.log('Incorreto!');
}

Pode ser escrito como:

estaCorreto ? console.log('Correto!') : console.log('Incorreto!');

Switch

Palavras reservadas: switch, case, break e default; exemplo:

switch (fruta) {
  case 'tomate':
    console.log('Tomates são R$0.49');
    break;
  case 'lima':
    console.log('Limas são R$1.49');
    break;
  case 'papaia':
    console.log('Papaias são R$1.29');
    break;
  default:
    console.log('Fruta desconhecida');
}
